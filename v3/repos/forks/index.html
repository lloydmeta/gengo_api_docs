Repo Forks API

List forks

GET /repos/:user/:repo/forks

Parameters

sort
newest, oldest, watchers, default: newest.

Create a fork

Create a fork for the authenticated user.

POST /repos/:user/:repo/forks

Parameters

org
Optional String - Organization login. The repository will be forked into this organization.

Response

Forking a Repository happens asynchronously. Therefore, you may have to wait a short period before accessing the git objects. If this takes longer than 5 minutes, be sure to contact Support.
